# 🚀 客户数据多源整合与 AI 洞察系统 (Demo)

## 📖 项目背景
本项目旨在解决当前客户数据分散在 **个人微信、企业微信、邮件、CRM** 等多个孤岛的问题。通过 ETL 流程清洗数据，并利用 AI 构建本地知识库，实现精准的客户画像分析与销售策略推荐。

## 🛠️ 核心流程
1. **数据采集 (Extract)**: 使用 `WeChatMsg` / `MailStore` 提取多源非结构化数据。
2. **清洗标准化 (Transform)**: 利用 `Power Query` 进行字段对齐、去重与隐私脱敏。
3. **AI 分析 (Analyze)**: 投喂至本地大模型 (RAG)，生成分析报告。

## 📂 文件说明
- `index.html`: **系统原型演示**。展示了客户画像、沟通摘要、课程推荐三个核心功能模块。
- `flowchart.html`: **业务流程图**。详细展示了从数据源到最终价值输出的技术路径。
